# PersonalBLOG

PersonalBLOG is a full-stack blog web application built with Node.js, Express, and MongoDB.  
The project provides user authentication, role-based access control, blog post management, and a REST API.

Live demo: https://personalblog-kruc.onrender.com  
GitHub repository: https://github.com/arman381/PersonalBLOG

---

## Features

- User registration and login using JWT
- Password hashing and secure authentication
- Role-based access (admin and user)
- Create, read, update, and delete blog posts
- Admin panel for managing users and posts
- RESTful API architecture
- MongoDB database integration
- Cloud deployment on Render

---

## Tech Stack

Backend:
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT
- bcryptjs
- dotenv
- cors

Frontend:
- HTML
- CSS
- Vanilla JavaScript

Deployment:
- Render
